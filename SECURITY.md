# Security Policy

## Supported Versions

We currently support security updates for the latest released version of this project.

| Version | Supported |
| ------- | --------- |
| latest  | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately.

Do not open a public GitHub issue for security vulnerabilities.

Please report the vulnerability via GitHub Private Vulnerability Reporting.

Include as much detail as possible:

- Affected version or commit
- Steps to reproduce
- Potential impact
- Suggested fix, if known

We aim to acknowledge reports within 72 hours and provide updates as we investigate.

## Disclosure Policy

Please allow us reasonable time to investigate and patch the issue before public disclosure.
