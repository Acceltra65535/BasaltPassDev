# Code of Conduct

## Our Pledge

We want BasaltPass to be a respectful, professional, and productive open source
community. Everyone participating in this project is expected to help maintain a
welcoming environment for contributors with different backgrounds, experience
levels, and perspectives.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, but be clear when something is incorrect or unsafe.
- Focus feedback on the work, not the person.
- Be patient with questions and precise with technical criticism.
- Respect maintainers' decisions on project scope, security posture, and release
  readiness.

## Unacceptable Behavior

- Harassment, threats, insults, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Repeated off-topic disruption of issues, discussions, or pull requests.
- Pressuring maintainers to disclose or rush security-sensitive work.

## Enforcement

Project maintainers may remove comments, close issues, reject contributions, or
block participants whose behavior violates this code of conduct.

Serious or repeated violations may result in a permanent ban from project spaces.

## Reporting

If you experience or observe unacceptable behavior, contact the project
maintainers privately. For security vulnerabilities, follow `SECURITY.md`
instead of using a public issue.

Reports will be reviewed as privately and fairly as practical.
